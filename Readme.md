# Overview of Libaraies Venv and Tabulate

## Pip

- pip is the standerd package manager for python
- used to install and manage libraries and dependencies from the Python Package Index

## Venv - Sets up a virtual enviroment

- a fresh copy of the latest edition of your python
- when you install and activate a virtual environment, all libraries insttalled with pip are local to that project
- avoids version conflicts
- generally a best practice

## Tabulate

- a lightweight tool for pretty-printing tabular data in various formats. It is commonly used to transform raw data structures like lists or dictionaries into readable tables for terminal output, Markdown, or HTML.

## To select the python interpreter in VS Code:

- Ctrl + Shift + P
